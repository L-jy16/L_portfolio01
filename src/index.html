WHat I do

javascript

JavaScript를 이용하여 동적 웹페이지를 개발하며, 사용자와의 상호작용을 향상시키는데 주력합니다. 다양한 기능과 동적 요소를 통해 웹 경험을 더욱 흥미롭게 만들어냅니다.

react

React를 활용하여 모던하고 성능 최적화된 웹 애플리케이션을 구축합니다. 컴포넌트 기반 아키텍처로 효율적이고 확장 가능한 사용자 경험을 제공합니다.

php

PHP를 활용하여 동적이고 효율적인 웹 애플리케이션을 개발합니다. 서버 측 스크립트 언어로써, 데이터베이스와의 통합 및 다양한 웹 기능을 구현하는 것에 중점을 둡니다.

node.js

Node.js를 이용하여 서버 측 JavaScript 환경에서 효율적이고 확장 가능한 웹 애플리케이션을 구축합니다. 비동기 프로그래밍으로 높은 성능을 유지합니다.
